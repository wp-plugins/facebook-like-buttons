== Description ==

The facebook like buttons plugin is a free add-on for wordpress and allows you to add the facebook like button in your wordpress/buddypress website.

== Installation ==
1. Download and upload the file `facebook-like-buttons.zip` from the plugin panel or unzip the file and upload the folder `facebook-like-buttons` to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` panel in your WordPress Site.
3. Drag and drop the facebook like button as a widget from your WordPress Site.
